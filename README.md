# LuERP

本项目最初是完成企业 供应商询价项目的平台 配合了 SENDWEIXIN开源等自动发邮件通知 供应商报价

之后 我们又添加了企业其他管理数据 完成了各个数据库 信息的数据集成

用Django框架开发 不再维护
